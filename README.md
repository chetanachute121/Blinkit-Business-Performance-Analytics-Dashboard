# Blinkit Business Performance Analytics Dashboard

## Project Description
This project is a Power BI dashboard created to analyze Blinkit's business performance.
It provides insights into sales, items, outlet performance, ratings, and customer trends.

## Key Metrics
- Total Sales
- Average Sales
- Number of Items
- Average Rating

## Dashboard Insights
- Sales by outlet type and size
- Item type analysis
- Outlet location performance
- Year-wise outlet establishment trend

## Tools Used
- Power BI
- Data Visualization
- Business Analytics
